📘 初一 · 一元一次方程

核心概念 解题步骤 典型例题

1. 什么是一元一次方程？

  • 一元：只有一个未知数（通常用 x 表示）。
  • 一次：未知数的次数是 1（没有 x²、x³）。
  • 方程：等号左右两边是相等的式子。

常见形式：

  • ax + b = c
  • ax + b = cx + d

2. 通用解题步骤

遇到一元一次方程，可以按下面 4 步来：

① 去括号、去分母。
② 把含 x 的项移到一边，数字项移到另一边。
③ 合并同类项。
④ 两边同时除以 x 的系数，得到 x 的值。

3. 典型例题

例 1：解方程：3x − 5 = 16

  1. 把 −5 移到右边：
    3x = 16 + 5 = 21
  2. 两边同除以 3：
    x = 21 ÷ 3 = 7

答：x = 7。

例 2：解方程：5x − 4 = x + 12

  1. 把含 x 的项移到左边：
    5x − x = 12 + 4
  2. 化简：
    4x = 16
  3. 两边同除以 4：
    x = 4

答：x = 4。

4. 小练习（可以让孩子自己先做）

  1. 2x + 3 = 11
  2. 4x + 6 = 2x + 16
  3. 7x − 5 = 3x + 15
👉 展开查看参考答案
  • 第 1 题：2x + 3 = 11 → 2x = 8 → x = 4
  • 第 2 题：4x + 6 = 2x + 16 → 2x = 10 → x = 5
  • 第 3 题：7x − 5 = 3x + 15 → 4x = 20 → x = 5
